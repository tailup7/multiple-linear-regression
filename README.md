| AveRooms | 地区の住宅の平均部屋数 |

| population | 地区の総人口 |
| AveOccup | 地区の1世帯あたりの平均人数 |

| Longitude | 地区の経度 |
| MedHouseValue | 地区の住宅価格の中央値 |
